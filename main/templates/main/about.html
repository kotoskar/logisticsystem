О нас

Способности системы:

  • Основной задачей продукта является определение местоположения аппаратов в реальном времени. С помощь этой системы беспилотники могут использоваться где угодно и работает везде.
  • Для отображения всех необходимых данных используется Web - страница, это позволит пользователю удаленно от базовой станции иметь доступ ко всем данным.
  • В нашей системе поддерживается одновременное слежение за 8 дронами, для каждого из них необходимо передать широту и долготу с хорошей точностью, а также давление.

О системе:

  • Для определения точного местоположения используется GPS и Глонасс, точность в средем (2-4 м). Данные передаются с помощью способа передачи LPWAN и LoRa модулей
  • Схема приёмника состоит из: Raspberry Pi 3 model B+ 4GB, GPS ATGM336H, BMP280, LoRa SX1278.
  • Схема передатчика состоит из: Raspberry Pi Pico, GPS ATGM336H, BMP280, LoRa SX1278.
